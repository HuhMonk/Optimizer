# HuhMan Optimizer | Does actually change settings

## A optimization tool for VR Games to make the game run better

### Features
Optimization Window Covering ( Graphics & Quality Settings, Particle System Settings, Physics Settings, Camera Settings, Mesh Merging, Other [only has audio quality rn])

Runtime Optimizer
which optimizes at runtime by settins max fps & other
